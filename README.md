# ✋ Hand-Controlled 3D Particle System

An interactive 3D particle visualization system controlled using hand gestures.  
Built using **Three.js** to create dynamic, real-time animations.

---

## 🚀 Live Demo

👉 https://hand-controlled-3d-system.vercel.app/

---

## 🚀 Features

- ✋ Hand gesture-based interaction  
- 🌐 Real-time 3D particle rendering  
- 🔵 Multiple particle shapes (Sphere, Heart, etc.)  
- ⚡ Smooth animations and transitions  
- 📱 Responsive and lightweight  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript 
- Three.js  

---

## 🎮 How It Works

- Uses gesture-based logic to control particle movement  
- Particles dynamically rearrange into different shapes  
- Real-time rendering using Three.js
